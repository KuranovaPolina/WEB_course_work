Квадратное уравнение — это уравнение вида \(ax^2 + bx + c = 0\), где a, b и с — некоторые коэффициенты, причём \(a \ne 0\).
Уравнение в которам a = 0 называется линейным и имеет вид bx + c = 0.

Решить квадратное уравнение, значит найти его корни. Это можно сделать с помощью дискримината.
Дискриминат вычисляется по формуле \(D = b^2 - 4ac\).

Дискриминат может принять 3 вида значений:
\(D > 0\), в таком случае уравнение имеет 2 вещественных корня, вычисляемые по формуле: $${x}_{1,2} = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$
\(D = 0\), в таком случае корень 1: $${x} = {-b \over 2a}.$$
\(D < 0\), в таком случае вещественных корней у квадратного уравнения нет.

Корень линейного уравнения высчитывается по формуле: $${x} = {-c \over b}.$$
